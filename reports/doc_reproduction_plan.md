# 主被动微波冰川探测文档理解与复现计划

## 这篇文档在讲什么

这篇文档不是一个普通的机器学习表格回归任务。它讲的是面向冰盖/冰川深层探测的主被动微波联合遥感方案。

核心目标是通过 P-L 波段的主动雷达和被动辐射计，反演冰盖内部的多种物理参数：

```text
密度廓线
介电常数廓线
温度廓线
冰盖内部结构
冰底/内部层反射特征
```

可以把它理解成：

```text
被动微波亮温负责看整体辐射响应
主动雷达回波负责看内部结构和衰减
两者联合起来约束冰盖内部物理参数
```

## 主动和被动分别是什么意思

### 被动微波

被动微波就是接收冰盖自己发出来的微波辐射，观测量通常是亮温。

文档里强调的是 P-L 波段宽带亮温。不同频率对不同深度的敏感性不同，所以宽带亮温可以提供深度方向上的信息。

### 主动微波

主动微波就是雷达自己发射电磁波，再接收回波。

雷达回波主要反映冰盖内部的介电常数突变、分层结构、冰底反射和传播衰减。

## 为什么要主被动联合

只看被动亮温时，亮温是一个积分结果，很多深度的信息混在一起，不容易区分到底是哪一层导致的变化。

只看主动雷达时，雷达能看到结构和衰减，但不一定能单独稳定反演温度、密度等连续廓线。

所以文档的逻辑是：

```text
雷达回波提供结构和衰减约束
被动亮温提供宽带辐射观测
联合反演冰盖内部温度、密度等参数
```

## 和我现在的数据是不是一回事

不是一回事。

你现在的数据是海冰/积雪厚度反演数据，主要包括：

```text
18-89 GHz 亮温通道
浮标雪深
位置、时间、气象变量
```

这适合做：

```text
海冰积雪厚度的被动微波统计反演
```

而文档里的复现需要：

```text
P-L 波段低频亮温
探冰雷达回波剖面
冰芯密度或钻孔温度廓线
冰盖厚度和分层信息
```

所以当前数据不能直接完整复现这篇文档。可以做的是：

```text
先用简化模型复现主被动联合探测的机理链路
再把你当前数据作为被动亮温雪深反演的类比实验
```

## 我现在做的最小复现

当前仓库新增了一个简化 demo：

```powershell
python active_passive_microwave\reproduce_doc_demo.py
```

它会生成：

```text
reports/active_passive_microwave_demo/
```

里面包括：

```text
density_dielectric_profile.png
passive_depth_response.png
radar_echo_profile.png
temperature_inversion.png
reproduction_notes.md
```

这个 demo 复现的是文档的核心逻辑：

```text
密度廓线 -> 介电常数
温度廓线 -> 被动亮温
介电常数分层 -> 主动雷达回波
被动亮温 + 雷达衰减 -> 温度廓线反演
```
